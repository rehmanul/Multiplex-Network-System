# Multiplex Analytics Engine

Python analytics engine for multiplex political-institutional networks.

## Features

- Signed network analysis (frustration index, structural balance)
- Multi-layer centrality metrics
- Multiplex PageRank
- Advanced institutional metrics

## Installation

```bash
pip install .
```

## Usage

```bash
python -m uvicorn src.main:app --host 0.0.0.0 --port 8001
```
